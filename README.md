## 操作指令

**'{keyword} {query}'**

**'dark {query}'**

**'dark on'			深色**
**'dark off'	    浅色**
**'dark'			深色 / 浅色**

## 快捷键

**shift ⇧ + ⌃ + M     深色 / 浅色**
